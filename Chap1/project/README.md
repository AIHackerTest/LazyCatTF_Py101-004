### CLI版天气通 weather.py

- 在终端中输入 python weather.py weather_info.txt 运行该程序

- 分别尝试用 for 和 while 循环将天气数据逐行录入 dictionary，将其中一个循环注释掉了

- 在 if statement 之前加一行 while True 实现多次查询

- 查询历史功能使用的是字符串 + 号拼接

- 加入了欢迎语和用户输入非关键词的情况

- 输入 h 或者 help 获取使用帮助
